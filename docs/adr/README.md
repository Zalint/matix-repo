## Convention

- Numérotation séquentielle, jamais réutilisée.
- Statuts : `Proposed` → `Accepted` → (éventuellement) `Superseded by ADR-XXXX`.
- Une ADR n'est **jamais modifiée** une fois `Accepted` ; on en crée une nouvelle qui la supersede.
